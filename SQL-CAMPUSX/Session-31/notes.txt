What is SQL?
=============

Ans: SQL(Structured Query Language) is a programming language used for managing and manipulating data in relational databases.
     It allows you to insert, update, retrieve, and delete data in a database. It is widely used for data management in
     many applications, websites, and businesses. In simple terms, SQL is used to communicate with and control databases.

Data Integrity
==============

Data Integrity in databases refers to the accuracy, completeness, and consistency of the data stored in a database.

It is a measure of the reliability and trustworthiness of data and ensures that the data is protected from errors, corruption, and unauthorized changes.

Methods to Ensure Data Integrity
                        DATA INTEGRITY
                              │
              ┌───────────────┼───────────────┐
              │               │               │
         Constraints     Transactions    Normalization
              │               │               │
        Apply rules       Operations      Minimize data
       and conditions    treated as one    redundancy &
        on the data       unit of work     ensure data
                                           consistency
1. Constraints

Constraints are rules or conditions that must be met for data to be inserted, updated, or deleted in a database table.

They are used to enforce the integrity of data and prevent data from becoming inconsistent or corrupted.

2. Transactions

A Transaction is a sequence of database operations that are treated as a single unit of work.

3. Normalization

Normalization is a database design technique that minimizes data redundancy and ensures data consistency by organizing data into separate tables.

Quick Revision: Data Integrity = Accuracy + Completeness + Consistency + Reliability of Data

Referential actions specify what should happen to related data when a referenced row is updated or deleted.

RESTRICT
CASCADE
SET NULL

ALTER TABLE Command
====================

The ALTER TABLE statement in SQL is used to modify the structure of an existing table. Some of the things that can be done using the ALTER TABLE statement include:

Adding new columns
Deleting existing columns
Modifying existing columns
                       ALTER TABLE
                            │
            ┌───────────────┼───────────────┐
            │               │               │
       ADD COLUMN      DROP COLUMN      MODIFY COLUMN
            │               │               │
        Add a new        Delete an         Modify an
          column      existing column   existing column

In simple words: ALTER TABLE is used to make changes to the structure of a table that has already been created.
